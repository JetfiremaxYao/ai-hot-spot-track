# Workflow Reference

## 1. 关键词监控工作流
1. 拉取关键词列表。
2. 按用户意图执行新增/启停/删除。
3. 返回更新后的统计摘要。

## 2. 热点刷新工作流
1. 启动刷新任务。
2. 记录任务启动结果。
3. 引导用户等待 `refresh:done` 或执行列表查询。

## 3. 热点查询工作流
1. 解析筛选参数。
2. 查询热点列表或详情。
3. 必要时执行已读/收藏/点赞写操作。
4. 输出分页信息与下一步动作。

## 4. 搜索工作流
1. 校验关键词非空。
2. `db` 或 `live` 模式检索。
3. 输出结果数量、来源分布、监控建议。

## 5. 策略与通知工作流
1. 查询当前策略。
2. 按最小变更更新策略。
3. 触发测试邮件并汇总结果。

## 推荐返回摘要模板
- 成功：`已完成 {动作}，共处理 {N} 条数据。`
- 失败：`{动作} 失败，原因: {reason}。建议: {fix}`
