# AI热点监控系统 - 项目总结

## 📋 项目概览

**项目名称**: AI热点监控系统 (AI Hot Spot)  
**项目目标**: 构建智能热点发现和监控平台，帮助用户第一时间获取AI领域最新动态  
**开发周期**: 2-3周 (分2阶段)  
**技术栈**: React 19 + Vite + TailwindCSS + Node.js + Express + SQLite + Prisma + OpenRouter

---

## 🎯 核心功能

### 第一阶段：网页版MVP
1. **关键词管理** - 添加、激活、暂停、删除关键词
2. **AI热点观测** - 自动爬取、分析、去重、推送
3. **多维度查询** - 筛选、排序、搜索热点
4. **实时推送** - WebSocket推送高热度热点
5. **响应式UI** - 独特美观的前端设计

### 第二阶段：Agent Skills（已完成）
6. **技能封装** - 已完成总控技能封装，见 `skills/ai-hotspot-orchestrator/SKILL.md`

## 🏗️ 架构设计

### 整体流程

```
┌─────────────────────────────────────────────────────────────┐
│ 前端 (React 19 + Vite + TailwindCSS)                        │
│ - 关键词管理、热点浏览、实时推送、筛选排序                   │
└────────────────┬────────────────────────────────────────────┘
                 │ REST API + WebSocket
┌────────────────▼────────────────────────────────────────────┐
│ 后端 (Node.js + Express 5.x)                                │
│ - 关键词API、热点API、WebSocket服务                          │
└────────────────┬────────────────────────────────────────────┘
                 │
        ┌────────┴────────┬────────────┐
        │                 │            │
┌───────▼───────┐ ┌──────▼──────┐ ┌──▼─────────┐
│ 爬虫服务      │ │ AI分析      │ │ 定时任务  │
│ (Cheerio)     │ │ (OpenRouter)│ │(node-cron)│
└───────┬───────┘ └──────┬──────┘ └──┬─────────┘
        │                │            │
        └────────────────┼────────────┘
                         │
                  ┌──────▼──────┐
                  │ 数据库       │
                  │(SQLite+     │
                  │Prisma)      │
                  └─────────────┘
```

### 关键技术决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 数据库 | SQLite | 轻量化，无需额外服务 |
| ORM | Prisma | 类型安全，迁移管理 |
| AI能力 | OpenRouter | 统一API，多模型支持 |
| 爬虫 | Cheerio | 轻量，性能好 |
| 实时通信 | Socket.io | 成熟稳定，支持降级 |
| 前端构建 | Vite | 极速HMR，开发体验最佳 |

## 🎨 前端设计理念

为了确保页面"足够独特，不千篇一律"，建议实现以下设计元素：

### 设计方向
- **配色**: 深色主题 + 渐变色强调（紫蓝-青绿）
- **排版**: 大字体标题 + 清晰的信息层级
- **动画**: 流畅的过渡 + 微交互反馈
- **布局**: 非对称布局 + 充分留白

### 推荐Shadcn/UI组件
- `Button` - 丰富的变体
- `Card` - 热点卡片容器
- `Badge` - 评分和标签
- `Dialog` - 详情弹窗
- `Select` - 筛选下拉
- `Tabs` - 多视图切换
- `Toast` - 实时通知
